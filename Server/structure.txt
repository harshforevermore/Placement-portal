SERVER DIRECTORY STRUCTURE
==========================

├── .gitignore                     # Git ignore file

│   ├── jwt.js                    # JWT configuration

│
├── models/                        # Database models (organized by user type)
│   ├── admin/

│   │   └── AdminLog.js           # Admin activity logs

│       ├── StudentProfile.js     # Extended student profile
